# WordAAL
Interactive Wordle-like demo the resulting strategies from the paper ['Playing Wordle with Uppaal Stratego' by Peter G. Jensen, Kim G. Larsen, and Marius Mikuƒçionis.](https://link.springer.com/chapter/10.1007/978-3-031-15629-8_15)


## Dev setup
Not using SvelteKit, so no need to install anything globaly. Just run the following commands in the `WordAAL-svelte/` folder:
```
npm install
npm run dev
```
